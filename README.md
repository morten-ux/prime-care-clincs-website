# Prime Care Clinics — Website

A simple landing page for Prime Care Clinics, a cosmetic and aesthetic treatment clinic.

## Structure

```
prime-care-clincs-website/
└── index.html      # Main landing page
└── README.md       # This file
```

## Sections

- **Nav** — Logo, links, and a Book Now CTA
- **Hero** — Headline, subtext, and consultation CTA
- **Services** — Four treatment cards (anti-wrinkle, fillers, skin rejuvenation, consultations)
- **About** — Brief clinic description and stats (years, clients, locations)
- **CTA** — Email contact / booking prompt
- **Footer** — Copyright, privacy/terms links

## Getting Started

Open `index.html` directly in a browser — no build step or dependencies required.

## Customisation Notes

- Update the `mailto:` link in the CTA section with your real contact email.
- Replace the stats (10+ years, 2k+ clients, 3 locations) with accurate figures.
- Add real clinic locations and a contact form when ready to expand the site.
